# Superstore Sales Analysis
An end-to-end sales performance analysis project using Excel, Power Query and Power BI.
## Project Overview

This is my first end-to-end data analytics portfolio project.

I worked on the Sample Superstore dataset to understand the company's sales performance, identify areas of profit and loss and uncover insights that could help improve business decisions.

I cleaned and prepared the data using Microsoft Excel and Power Query before building an interactive dashboard in Power BI. I also documented the entire project in a report, explaining my approach, findings and recommendations.
## Project Objectives

The goal of this project was to:
- Understand the company's sales performance.
- Identify the most profitable products and categories.
- Compare sales across different regions and states.
- Analyze sales trends over time.
- Build an interactive dashboard that allows users to explore the data.
- Make practical recommendations based on the analysis.

## Key Insights

- The company generated over **$2.29 million** in sales.
- The **West Region** recorded the highest sales.
- California was the best-performing state.
- The **Technology** category generated the highest revenue.
- **Tables** recorded the highest loss despite generating sales.
- Sales were generally highest between **September and November**, showing a clear seasonal trend.
## Recommendations

Based on the analysis, I recommended that the company should:
- Increase inventory before the peak sales period.
- Review the pricing and discount strategy for Tables.
- Improve the profitability of Bookcases.
- Invest more in high-performing cities and regions.
- Continue providing efficient Standard Class shipping while exploring opportunities to improve premium shipping options.
